# سالویا فری – SalviaFeri

## 🚀 معرفی
**SalviaFeri** ابزاری پیشرفته بر بستر GitHub Actions برای دانلود از هر سایت (لینک مستقیم، یوتیوب، آپارات، اینستاگرام) با خروجی ZIP، رمزگذاری، تقسیم خودکار، و مدیریت هوشمند فضا.

## ✨ ویژگی‌ها
- پشتیبانی از هر سایتی
- خروجی ZIP با رمز دلخواه
- هر لینک یک فایل جداگانه
- فقط فایل‌های زیر ۱ گیگابایت
- تقسیم خودکار فایل‌های بزرگ به پارت‌های ۹۰ مگابایتی
- حذف خودکار قدیمی‌ترین فایل در صورت پر شدن فضا (بیشتر از ۹۰۰ مگابایت)
- پاکسازی دستی با یک کلیک
- رابط دانلود با index.html

## 📖 راهنمای استفاده
1. این ریپازیتوری را **Fork** کنید.
2. به تب **Actions** بروید.
3. روی **SalviaFeri Downloader** کلیک کنید.
4. دکمه **Run workflow** را بزنید.
5. لینک(ها) و رمز دلخواه (اختیاری) را وارد کنید.
6. پس از اتمام، به پوشه **Downloads** رفته و فایل را دانلود کنید.

## 🧹 پاکسازی
برای حذف همه فایل‌ها: **Actions** → **Cleanup Downloads - SalviaFeri** → **Run workflow**

## ⚠️ محدودیت‌ها
- حداکثر حجم هر فایل: **۱ گیگابایت**
- حجم کل پوشه Downloads همیشه زیر **۹۰۰ مگابایت** نگه داشته می‌شود.
